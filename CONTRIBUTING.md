# 贡献指南

感谢你对 WebTape 的关注！

## 许可协议声明

WebTape 基于 **Apache License 2.0** 发布。

**提交 Pull Request 即表示你同意**：你的贡献将以 Apache License 2.0 授权给本项目，项目维护者有权在该协议框架内使用、修改和分发你的贡献。

## 禁止用途说明

请勿通过贡献代码或文档来引导用户将 WebTape 用于：

- 未经目标系统所有者授权的网络访问或数据采集
- 违反目标网站服务条款的自动化操作
- 任何商业性质的爬虫、数据售卖或 API 代理服务

## 如何贡献

1. Fork 本仓库
2. 基于 `main` 创建功能分支
3. 提交代码，保持提交信息清晰
4. 发起 Pull Request，描述你的改动目的

## 行为准则

参与本项目即表示你同意以专业、友善的方式与社区成员互动。
